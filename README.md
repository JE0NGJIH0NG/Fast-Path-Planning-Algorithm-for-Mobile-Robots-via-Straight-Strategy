# Fast-Path-Planning-Algorithm-for-Mobile-Robots-via-Straight-Strategy
This repository contains the code used in the paper Fast Path-Planning Algorithm for Mobile Robots via Straight Strategy.

The paper was published on February 15, 2026.
- https://www.mdpi.com/2076-3417/16/4/1952

The algorithms used in the experiments are Q-RRT*, F-RRT*, MQ-RRT*, and Straight-RRT.
